# Images Folder

This folder contains images for the course website.

## How to Add Images

1. Upload your image files to this folder (JPG, PNG, GIF, or SVG formats)
2. Reference them in your .md files using:

```
![Alt text describing image](/dh340ss26ai/assets/images/your-image-name.jpg)
```

## Image Guidelines

- Use descriptive alt text for accessibility
- Recommended sizes:
  - Full-width images: max 1200px wide
  - Inline images: max 600px wide
- Keep file sizes under 500KB for faster loading

## Current Placeholders

Students can replace placeholder images by:
1. Clicking "Add file" → "Upload files" in GitHub
2. Dragging images to this folder
3. Committing the changes
4. Updating the image paths in the .md files
